# OG share cards

Per-game 1200×630 PNG cards generated by `scraper/scripts/generate_og_cards.py`.

Used as:
- Featured image on WPR per-game WordPress articles (carries OG metadata
  via WP automatically)
- Manual paste-into-social by editors covering games on Facebook/X
- Future: embedded inside the widget for richer hero cards

The generator's cache key (`<game-id>.cache`) tracks the visible-on-card
fields (status, scores, playoff round, date). Re-running the generator
is cheap — only changed games are re-screenshot.

Files are committed alongside `data/` json updates so the GitHub Pages
deploy can serve them at:

  `https://rowanflynnpilot.github.io/wpr-prep-sports/data/og/<sport>/<game-id>.png`
